# 📦 Installation Guide - PK B2B Orders

## Quick Install (3 Steps)

### Step 1: Download
Download: **PK B2B Orders Setup 1.0.0.exe**

### Step 2: Bypass Windows Warning
1. Double-click the installer
2. Windows shows: **"Windows protected your PC"**
3. Click **"More info"**
4. Click **"Run anyway"**

**This is normal!** The app is safe.

### Step 3: Install
1. Click **Next**
2. Choose shortcuts (Desktop + Start Menu)
3. Click **Install**
4. Click **Finish**

**Done!** Launch from desktop shortcut.

---

## Why Windows Shows Warning?

The app doesn't have a $500/year digital signature certificate.

**This does NOT mean it's dangerous!**

Windows blocks ALL new apps without certificates.

---

## ✅ The App is Safe

- Built by your company
- Only connects to your order system
- No data collection
- No external connections

## Need Help?

If the installer won't run:
1. Right-click installer
2. Click **"Run as administrator"**
3. Follow steps above
