# Sound Files

Place your notification sound file here:
- **new-order.mp3** - Notification sound for new orders

You can download a free notification sound from:
- https://notificationsounds.com/
- https://freesound.org/

Or use any .mp3 file of your choice.

For now, the application will work without the sound file, but the audio notification will be skipped if the file is not found.
